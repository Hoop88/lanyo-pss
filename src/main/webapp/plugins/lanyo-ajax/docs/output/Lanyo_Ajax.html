Class Lanyo_Ajax

Package:Global
Defined In:core-3.2.js
Class:Lanyo_Ajax
Extends:Object
蓝源RIA框架基本设置，可以用来配置一些平台的功能特性 在项目中会根据实际应用情况使用apply等方法来定制。 
//例子：

Ext.apply(Lanyo_Ajax,{
    permissionCheck:false,
    extVersion:'ext-3.2',
    script:'/demo-script',
    PersonalityUrl:"#",
    formatUrl:function(baseUrl,cmd){
        return  baseUrl+"?cmd="+cmd;
    }
});


This class is a singleton and cannot be created directly.
