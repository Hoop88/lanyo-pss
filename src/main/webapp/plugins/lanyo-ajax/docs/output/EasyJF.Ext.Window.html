Class EasyJF.Ext.Window

Package:EasyJF
Defined In:core-3.2.js
Class:Ext.Window
Extends:Object
显示单例模式的Window,可将自己的面板放入到EasyJF.Ext.Window中。 
//示例：

//创建需要在窗口中显示的panel

var myshowpanel=new MyShowPanel();
//调用工厂方法show来显示窗口

EasyJF.Ext.Window.show({
	single:false,
	width:400,
	height:160,
	scope:this,
	title:"确认信息",
	buttons:['yes','no'],
	items:myshowpanel,
	handler:function(btn,win,fp){
		if(btn=='yes'){
			//my customer handler

		}else{
			win.hide();
		}
	}
});


This class is a singleton and cannot be created directly.

This class has no public properties.

MethodDefined By
 show( Object config ) : void
调用该方法来显示一个窗口。
调用该方法来显示一个窗口。
Parameters:
  • config : Object
    配置显示窗口的对象，包括以下配置项：
    • single {Boolean} : 设置是否是单例显示，如果是true，在关闭窗口后会删除窗口中的panel。如果要多次使用该窗口显示同一个panel，single要设置为true。
    • width {Integer} : 设置窗口的宽度
    • height {Integer} : 设置窗口高度
    • scope {Object} : 方法中按钮响应方法的执行域
    • title {String} : 窗口的显示名称
    • buttons {Array} : 自定义窗口下显示的按钮，可选值有：'ok','cancel','yes','no'
    • items {Object} ： 窗口内要显示的面板panel
    • handler {Function} : 窗口下按钮的响应事件,其中传入的参数有：
      • btn :点击的btn代码
      • win :当前window
      • fp ：窗口中的panel
    • layout {String} : 窗口的布局，默认为fit
    • buttonAlign {String} : 窗口下按钮的对齐方式，默认为'center'
    • closeAction {String} : 窗口关闭事件，默认为'hide'
Returns:
  • void
